# Coquitos Ice Cream at Le Lo Lai - Website

## 1. Project Description
Sitio web oficial para Coquitos Ice Cream at Le Lo Lai, una heladería familiar en Kissimmee, FL, famosa por sus mantecados, limbers y piraguas de coco, parcha, acerola y otros sabores tropicales. Público objetivo: familias locales y turistas en el área de Kissimmee/Orlando buscando postres helados auténticos.

## 2. Page Structure
- `/` - Home (single-page landing con secciones: Hero, About, Menú, Reseñas, Visítanos, Contacto)

## 3. Core Features
- [x] Diseño moderno con paleta tropical (coral mango, verde palma, crema coco)
- [x] Hero con imagen de fondo tropical
- [x] Sección de sabores destacados (Coco, Parcha, Acerola, Mantecados, Limbers, Piraguas)
- [x] Reseñas reales de clientes
- [x] Mapa con ubicación y horarios
- [x] Contacto directo por teléfono
- [x] Botones a Google Maps para direcciones

## 4. Data Model Design
No requiere base de datos por ahora (sitio informativo).

## 5. Backend / Third-party Integration Plan
- Google Maps embed para ubicación
